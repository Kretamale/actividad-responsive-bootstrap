# Comandos básicos de Git

## git --version

### Definición
Muestra la versión de Git instalada en el equipo.

### Ejemplo

```bash
git --version
```

---

## git init

### Definición
Inicializa un repositorio Git en la carpeta actual.

### Ejemplo

```bash
git init
```

---

## git add

### Definición
Agrega archivos al área de preparación (staging).

### Ejemplo

```bash
git add .
```

---

## git commit

### Definición
Registra los cambios realizados en el repositorio con un mensaje descriptivo.

### Ejemplo

```bash
git commit -m "Primer commit"
```

---

## git status

### Definición
Muestra el estado actual del repositorio.

### Ejemplo

```bash
git status
```

---

# Flujo típico de trabajo

```bash
git add .
git commit -m "mensaje"
git status
```
